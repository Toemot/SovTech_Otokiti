#Implementation of an OpenAPI (formerly Swagger)

#chuck

#search

## Supported user Actions

*Return all the joke categories of jokes for Chuck Noris
*Return all the Star Wars people
*Simultaneously search both the Chuck Norris and Star Wars API.

# Endpoints
*/api/chuck/categories GET
*/api/swapi/people GET
*/api/search GET (Call both and also contain metadata which indicates which API the result belongs to).
